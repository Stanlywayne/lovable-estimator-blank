
# BewBec Estimator

A React application built with Vite, TypeScript, and Tailwind CSS.

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components

## Development

This project uses Lovable for development with GitHub sync enabled.
